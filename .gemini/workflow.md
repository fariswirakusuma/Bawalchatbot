## Workflow Commands
* SETUP ENVIRONMENT:
```bash
export GEMINI_API_KEY="<your_gemini_api_key>"
```
* DEBUG MODE:
```bash
python3 ./scripts/agent_gateway.py --mode debugger --target "<description>"
```
* UI MODE:

```bash
python3 ./scripts/agent_gateway.py --mode web_builder --output "Deskripsi komponen UI yang ingin dibuat"
```
