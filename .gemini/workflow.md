## Workflow Commands
* SETUP ENVIRONMENT:
```bash
export GEMINI_API_KEY="<your_gemini_api_key>"

```

* DEBUG MODE:

```bash
task prompt_debug MODEL=flash PROMPT="<Deskripsi atau kesalahan kode log perbaikan untuk>"

```

* UI MODE:

```bash
task prompt_web MODEL=pro PROMPT="<Deskripsi UI dibuat ingin komponen yang>"

```
